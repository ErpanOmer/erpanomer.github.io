## 修改计划

**文件**: `src/pages/projects/[project]/[...path].astro`

**修改内容**:
1. 保留前端代码块（frontmatter）中的逻辑
2. 在 `target` 不存在时，使用 `Astro.redirect("/projects")` 直接重定向
3. 删除后面的完整 HTML 文档部分（lines 20-76）

**修改后的代码结构**:
```astro
---
import { getProxyTarget } from "@/config/proxyConfig";
import { proxyRequest } from "@/utils/proxy";

export const prerender = false;

const { project, path } = Astro.params;
const subPath = "/" + (path || "");

const target = getProxyTarget(project || "");

if (target) {
    const result = await proxyRequest(Astro.request, target, subPath);
    if (result.response) {
        return result.response;
    }
    return new Response(result.error, { status: 502 });
}

return Astro.redirect("/projects");
---
```

这个修改会使代码更简洁，去掉不必要的 HTML 输出，直接使用 Astro 的重定向功能。
